## 1. Example In-Flight Feature

**Theme:** Onboarding
**One-liner:** New users land on a tailored welcome screen within three seconds of signup.
**Status:** clarifying

### Acceptance criteria

| #   | Criterion |
|-----|-----------|
| 1.1 | User selects goals during signup |
| 1.2 | Welcome screen renders content based on selections |
| 1.3 | Fallback to generic content if no goals selected |
| 1.4 | Page loads in under 3 seconds |

- ML-based recommendations
- A/B testing
